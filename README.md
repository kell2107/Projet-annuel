# Fonctionnement 
Le but de ce jeu est de gagner soit en marquant des points indiquer sur la carte ou en prennant les pions adverses. Le plateau dispose de 14 pions (7 pions pour les deux joueurs). Le jeu oppose les pions bleu (joueur) contre les pions rouge (IA). Lorsqu'on lancera le jeu, il placera aléatoirement sur la carte 4 point à atteindre pour gagner la partie :

3 point doré = 1 point
1 point jaune = 3 point
